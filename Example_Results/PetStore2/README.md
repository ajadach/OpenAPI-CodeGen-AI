# PetStore2Client

API client library for **Swagger Petstore** â€” generated by Open API codegen by AI.

## Description

This library provides a Python interface for the Swagger Petstore REST API located at `https://petstore.swagger.io/v2`.  
It covers the following modules: **pet**, **store**, **user**.

## Requirements

- Python >= 3.12
- `requests`
- `pydantic`
- `robotframework` (for Robot Framework support)

## Installation

### As a pip package

```bash
pip install dist/PetStore2Client-1.0.0-py3-none-any.whl
```

### Local usage without pip

Add the `PetStore2` folder to `sys.path` in your script:

```python
import sys
sys.path.insert(0, "/path/to/PetStore2")

from PetStore2Client.client import PetStore2Client
client = PetStore2Client()
```

Or run your scripts from the `PetStore2` root directory so Python resolves imports correctly.

## Quick Start

```python
from PetStore2Client.client import PetStore2Client

client = PetStore2Client()

# Get a pet by ID
pet = client.pet.get_pet_by_id(1)
print(pet)

# Get store inventory
inventory = client.store.get_inventory()
print(inventory)
```

## Method Reference

### pet

- **`post_upload_file(pet_id: int, additional_metadata: Optional[str], file: Optional[Any]) -> dict`**  
  Uploads an image. Returns ApiResponse dict.

- **`post_add_pet(body: dict) -> None`**  
  Add a new pet to the store. body = Pet object.

- **`put_update_pet(body: dict) -> None`**  
  Update an existing pet. body = Pet object.

- **`get_find_pets_by_status(status: list) -> list`**  
  Finds Pets by status. status = list of `"available"`, `"pending"`, `"sold"`.

- **`get_find_pets_by_tags(tags: list) -> list`**  
  Finds Pets by tags. tags = list of tag strings.

- **`get_pet_by_id(pet_id: int) -> dict`**  
  Find pet by ID. Returns Pet object.

- **`post_update_pet_with_form(pet_id: int, name: Optional[str], status: Optional[str]) -> None`**  
  Updates a pet in the store with form data.

- **`delete_pet(pet_id: int, api_key: Optional[str]) -> None`**  
  Deletes a pet.

### store

- **`get_inventory() -> dict`**  
  Returns pet inventories by status. Returns map of status â†’ quantity.

- **`post_place_order(body: dict) -> dict`**  
  Place an order for a pet. Returns Order object.

- **`get_order_by_id(order_id: int) -> dict`**  
  Find purchase order by ID. Returns Order object.

- **`delete_order(order_id: int) -> None`**  
  Delete purchase order by ID.

### user

- **`post_create_users_with_list_input(body: list) -> None`**  
  Creates list of users with given input array.

- **`get_user_by_name(username: str) -> dict`**  
  Get user by user name. Returns User object.

- **`put_update_user(username: str, body: dict) -> None`**  
  Updated user.

- **`delete_user(username: str) -> None`**  
  Delete user.

- **`get_login_user(username: str, password: str) -> str`**  
  Logs user into the system. Returns login token.

- **`get_logout_user() -> None`**  
  Logs out current logged in user session.

- **`post_create_users_with_array_input(body: list) -> None`**  
  Creates list of users with given input array.

- **`post_create_user(body: dict) -> None`**  
  Create user.

## Environment / Configuration

The `base_url` parameter in the client constructor defaults to `https://petstore.swagger.io/v2`.  
Override it when instantiating the client:

```python
client = PetStore2Client(base_url="https://my-custom-petstore.example.com/v2")
```
